## Tentang Project Ini

Ini adalah repository untuk project "Inventaris Barang" yang dibangun menggunakan Laravel 12. Project ini bertujuan untuk memudahkan dalam mengelola barang, mencatat barang, melacak barang, serta membuat laporan barang yang jauh lebih siginifikan. 

- **Status:** "Selesai"
- **Usage:** "Untuk Pribadi"

## Fitur

- **Dashboard:** Menampilkan ringkasan data barang
- **Data Barang:** Menampilkan data barang
- **Data Kategori:** Menampilkan data kategori
- **Data Satuan:** Menampilkan data satuan
- **Data User:** Menampilkan data user
